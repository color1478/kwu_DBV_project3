# 관리자 계정 로그인 방법

## 방법 1: 비밀번호 업데이트 후 로그인 (권장)

### 1단계: MySQL에서 비밀번호 업데이트

MySQL Workbench 또는 터미널에서 다음 SQL을 실행하세요:

```sql
USE ddarungi_db;

UPDATE users 
SET password_hash = '$2b$10$ssnJJgHlzy/X2SBnUpa1eeEpBh0G31lZrObIRn3ONZvRqokBHfZie' 
WHERE email = 'admin@ddarungi.com';
```

또는 `UPDATE_ADMIN_PASSWORD.sql` 파일을 실행하세요.

### 2단계: 로그인

1. 브라우저에서 `http://localhost:3000/login` 접속
2. 다음 정보로 로그인:
   - **이메일**: `admin@ddarungi.com`
   - **비밀번호**: `admin123`

### 3단계: 관리자 페이지 접근

로그인 후 헤더의 "관리자" 링크를 클릭하거나, 직접 `http://localhost:3000/admin`으로 접속하세요.

---

## 방법 2: 새 계정 생성 후 역할 변경

### 1단계: 회원가입

1. `http://localhost:3000/register`에서 새 계정 생성
2. 원하는 이메일과 비밀번호 입력

### 2단계: MySQL에서 역할 변경

```sql
USE ddarungi_db;

-- 방금 생성한 계정의 이메일로 검색
SELECT user_id, email, role FROM users WHERE email = '생성한이메일@example.com';

-- 역할을 ADMIN으로 변경
UPDATE users SET role = 'ADMIN' WHERE email = '생성한이메일@example.com';
```

### 3단계: 다시 로그인

회원가입한 계정으로 로그인하면 관리자 권한으로 접근할 수 있습니다.

---

## 방법 3: 다른 비밀번호 사용하기

다른 비밀번호를 사용하고 싶다면:

```bash
# server 디렉토리에서
cd server
node scripts/generate-password-hash.js 원하는비밀번호
```

생성된 해시를 복사해서 MySQL에서 업데이트:

```sql
UPDATE users 
SET password_hash = '생성된_해시' 
WHERE email = 'admin@ddarungi.com';
```

## 확인

관리자 계정이 제대로 설정되었는지 확인:

```sql
SELECT user_id, email, nickname, role, is_active 
FROM users 
WHERE email = 'admin@ddarungi.com';
```

`role`이 `ADMIN`이고 `is_active`가 `1` (TRUE)이어야 합니다.
