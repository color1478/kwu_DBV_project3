# 기능 요약 및 위치 안내

## 관리자 기능 위치

### 1. 자전거 관리 (이용 중, 고장, 정비 중)
**위치**: `/admin/bikes`
- 자전거 목록 조회
- 상태 변경: 이용 가능 / 이용 중 / 고장 / 정비 중
- 대여소 배정

### 2. 신고 및 정비 관리
**위치**: `/admin/reports`
- 전체 신고 목록 조회
- 신고 검증 (유효/무효)
- 유지보수 주문 배정 (담당자, 우선순위, 마감일)
- 유지보수 상태 관리

### 3. 재배치 관리
**위치**: `/admin/rebalancing`
- 자전거 부족 대여소 식별
- 자전거 여유 대여소 식별
- 자동 재배치 경로 제안
- 재배치 실행

### 4. 대여소 이용률 지도
**위치**: `/admin/stations-map`
- 평소 이용률 대비 현재 자전거 배치율 색상 표시
- 지도에서 모든 대여소 확인
- 부하율별 분류

### 5. 상세 통계 대시보드
**위치**: `/admin/stats`
- 지역별 대여소 통계
- 자전거 상태별 분포
- 신고/유지보수 상세 통계
- D3.js 시각화

---

## 사용자 기능 위치

### 1. 혼잡도 예측
**위치**: `/congestion`
- 시간대별 대여 패턴 분석
- 대여소 혼잡도 예측
- 이용하기 좋은 대여소 추천

### 2. 고장 신고 업적 배지 (스탬프 형식)
**위치**: `/mypage` - "획득한 배지" 섹션
- 1회: 첫 신고 기여 🎉
- 3회: 현장 탐정 🔍
- 5회: 정비 파트너 🤝
- 10회: 신고 전문가 ⭐
- 20회: 신고 마스터 👑
- 배지 진행도 표시 (스탬프 형식)

### 3. 대여소 상세 - 자전거 목록
**위치**: `/station/[id]`
- 대여소에 등록된 자전거 목록
- 각 자전거의 상태 (이용 가능/이용 중/고장/정비 중)
- 자전거 ID 및 구매일

## 주요 개선 사항

### ✅ 완료된 작업

1. **부족 대여소 예시 추가**
   - 시드 데이터에서 일부 대여소의 bikes_available을 1-2대로 설정

2. **부하율 표시 색상 변경**
   - 회색 → 범례 색상 (빨강/주황/초록/파랑)
   - 부하율 배경색이 상태에 맞게 표시

3. **통계 대시보드 추가**
   - `/admin/stats` 페이지 생성
   - 지역별 통계, 자전거 상태별 분포 추가
   - D3.js 파이 차트로 시각화

4. **관리자 대시보드 개선**
   - 평소 이용률 대비 색상 표시 지도 (`/admin/stations-map`)
   - 재배치 자동 추천 개선 (`/admin/rebalancing`)

5. **사용자 기능 추가**
   - 혼잡도 예측 기능 (`/congestion`)
   - 고장 신고 업적 배지 시스템 (스탬프 형식)
   - 배지 진행도 표시

6. **대여소 상세 페이지**
   - 자전거 목록 및 상태 표시

## 접근 경로 요약

### 관리자
- 회원 관리: `/admin/users`
- 대여소 관리: `/admin/stations`
- **자전거 관리**: `/admin/bikes` ← 이용 중, 고장, 정비 중 관리
- **신고 및 정비 관리**: `/admin/reports` ← 신고 검증, 유지보수 배정
- **재배치 관리**: `/admin/rebalancing` ← 자동 재배치 추천
- 상세 통계: `/admin/stats`
- 이용률 지도: `/admin/stations-map`

### 사용자
- 대여소 탐색: `/`
- 대여소 상세: `/station/[id]` ← 자전거 목록 표시
- 혼잡도 예측: `/congestion` ← 시간대별 예측
- 마이페이지: `/mypage` ← 배지 (스탬프 형식)
